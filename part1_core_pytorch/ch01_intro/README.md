# Chapter 1: Introducing Deep Learning and the PyTorch Library

## Setup Instructions
1. **Create Virtual Environment** (As per Section 1.6):
```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
```

2. **Install Dependencies:**
```
pip install -r requirements.txt
```

3. **Verify Installation:**
```
python check_environment.py
```

<br>

## Notebook Usage
When creating a new Jupyter Notebook for this sprint:
- Copy the code from notebook_boilerplate.py into the first cell.
- Run the cell to ensure all imports work before proceeding.
